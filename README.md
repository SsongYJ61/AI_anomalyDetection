# 🧠 AI_anomalyDetection  
### 대교 데이터를 활용한 이상감지 프로젝트  

---

## 🎯 프로젝트 개요  

**목표:**  
> “다중 센서 데이터를 활용해 교량의 구조적 건전성을 실시간으로 모니터링하고,  
> 이상 징후를 조기 탐지할 수 있는 데이터 기반 모델을 구축한다.”

이 프로젝트는 교량에 부착된 다중 센서(가속도, 풍속, 온도, 변형률 등)의 시계열 데이터를 기반으로,  
**정상적인 구조 거동(Healthy State)**을 모델링하고,  
그 패턴이 깨질 때 **이상 상태(Anomaly)**를 탐지하는 것을 목표로 한다.

---

## 🧩 분석 가설 (Hypotheses)

| 가설 | 내용 | 검증 방법 |
|------|------|-----------|
| **H1** | 환경 요인(온도, 풍속 등)은 구조 반응(진동, 하중 등)에 유의미한 영향을 미친다. | Pearson 상관분석, 회귀분석 |
| **H2** | 정상 상태에서는 센서 간 상관관계가 일정하게 유지된다. | Rolling correlation 분석 |
| **H3** | 환경 요인으로 설명되지 않는 구조 반응의 잔차는 이상 징후를 의미한다. | Residual 분석 + SPRT 검정 |
| **H4** | 이상은 잔차·상관관계·변동성의 복합 패턴으로 나타난다. | Residual + ΔCorr + ΔVariance 지수화 |
| **H5** | 상관관계 붕괴는 구조적 손상 또는 센서 고장을 시사한다. | 센서 간 Correlation Network 변화 분석 |

---

## ⚙️ 데이터 개요  

- **데이터 출처:** 교량 구조물 센서 실측 데이터
- **센서 구성:**
  - `ACC` : 가속도 (Acceleration)
  - `CAC` : 하중 (Load / Pressure)
  - `FBG` : 광섬유 변형률 (Strain)
  - `FBT` : 온도 (Temperature)
  - `WGT` : 풍압 (Wind Pressure)
  - `WVR` : 풍속 (Wind Velocity)
  - `EQK` : 지진/진동 (Seismic)
- **데이터 특성:**
  - 시간 기반 시계열 (Datetime index)
  - 결측/단절 구간 존재
  - 센서별 단위 불일치 → 스케일링 필요
